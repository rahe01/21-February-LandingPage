# 🇧🇩 Bangla Heritage

A web-based project dedicated to celebrating and preserving the rich history and cultural significance of **International Mother Language Day**. This project was developed as a submission for the **Code For Language 2026** coding competition.

🚀 **Live Demo:** [banglaheritage.netlify.app](https://banglaheritage.netlify.app/)

---

## 📖 Project Overview

**Bangla Heritage** is an interactive platform designed to educate users about the Language Movement of 1952. The website focuses on the struggle for the Bangla language, the significance of the Shaheed Minar, and how these events led to the global recognition of International Mother Language Day by UNESCO.

### ✨ Key Features
* **Historical Timeline:** A detailed look at the pivotal events of 1952.
* **Cultural Insights:** Exploration of Bangla literature and heritage.
* **Responsive Design:** Optimized for viewing on desktops, tablets, and mobile devices.
* **Interactive UI:** Engaging elements to make learning about history more immersive.
